🚀 WORKFLOW - TEACHER CREATES EXAM
────────────────────────────────────

METHOD 1: MANUAL QUESTIONS
1. Click "Create Exam"
2. Enter Title, Topic, Grade, Duration, School
3. Select "Manual" mode
4. Add questions one by one (MCQ, True/False, Short Answer, Essay)
5. Edit questions as needed before publishing
6. Click "Publish Exam"
7. Exam saved with questions and options in database
8. Exam code and PIN generated automatically

METHOD 2: PRACTICAL PDF EXAM
1. Click "Create Exam"
2. Enter Title, Topic, Grade, Duration, School
3. Select "Upload PDF Practical" mode
4. Choose PDF file from computer
5. System uploads to Bunny CDN automatically
6. Click "Next" → "Publish Exam"
7. Exam saved with Bunny CDN URL
8. Students access practical exam via direct PDF link

METHOD 3: AI-GENERATED QUESTIONS
1. Click "Create Exam"
2. Enter Title, Topic, Grade, Duration, School
3. Select "Generate with AI" mode
4. Specify number of questions and difficulty
5. AI generates questions automatically
6. Edit any questions as needed
7. Click "Publish Exam"
8. Exam saved with AI-generated questions

🔄 WORKFLOW - TEACHER EDITS EXAM
──────────────────────────────────
1. Go to Exams Dashboard
2. Click "View" on exam to open viewer
3. Click "Edit" button
4. Form pre-fills with existing exam data
5. Questions load with all options
6. Edit title, topic, grade, duration, school
7. Edit or delete individual questions
8. Add new questions if needed
9. Click "Publish Exam" to save changes
10. Database UPDATEs exam and refreshes questions

🗑️ WORKFLOW - TEACHER DELETES EXAM
────────────────────────────────────
1. Go to Exams Dashboard
2. Click "View" on exam
3. Click "Delete" button
4. Confirm deletion
5. System cascades:
   - DELETE all options for all questions
   - DELETE all questions for exam
   - DELETE exam record itself
6. All done in transaction (all-or-nothing)

📊 DATABASE SCHEMA
───────────────────
exams:
  - exam_id (PK)
  - title, topic, grade, duration
  - school_id (FK to schools)
  - exam_code, pin (auto-generated)
  - status, created_by
  - start_time, end_time, is_active

questions:
  - question_id (PK)
  - exam_id (FK)
  - question_text
  - question_type (mcq|essay|true_false|practical)
  - marks

options:
  - option_id (PK)
  - question_id (FK)
  - option_text (contains Bunny URL for practical)
  - is_correct

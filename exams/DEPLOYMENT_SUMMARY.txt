DATABASE SCHEMA (CORRECT)
Table: exams
├─ exam_code (VARCHAR) - Unique code for student entry
├─ title (VARCHAR) - Exam title
├─ topic (VARCHAR) - Subject/topic
├─ grade (VARCHAR) - Grade level
├─ status (ENUM) - draft/active
├─ start_time (DATETIME) - When exam started (NULL if draft)
├─ duration (INT) - Minutes
├─ created_by (INT) - User ID
└─ created_at (TIMESTAMP) - Auto timestamp

Table: exam_questions
├─ question_id (PK)
├─ exam_id (FK)
├─ question_text (TEXT)
├─ question_type (ENUM) - multiple_choice/true_false/short_answer
└─ points (INT)

Table: exam_options (for MC & TF)
├─ option_id (PK)
├─ question_id (FK)
├─ option_text (TEXT)
└─ is_correct (BOOLEAN)

Table: exam_answers (for SA)
├─ answer_id (PK)
├─ question_id (FK)
└─ answer_text (TEXT)
